# Saifi — Image Generation Prompts

**How to use:** Each prompt below is complete and self-contained — the consistent style suffix is already baked in. Copy one block, paste into your generator (Higgsfield / Midjourney / etc.), generate, then save to `public/images/` using the exact filename shown.

**Why these look real (anti-AI):** every prompt forces 35mm full-frame, natural daylight, warm summer tones, slight film grain, candid real people, and explicitly bans text/logos/watermarks. Keep the whole set consistent so it reads as one photo shoot.

**Save location:** `public/images/<filename>.jpg`
**Priority order:** event covers first → hero → cities → category tiles → lifestyle/guide.
